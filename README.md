# Práctica: Introducción a GitHub y Azure

## Descripción

Esta práctica tiene como objetivo explicar los conceptos básicos de GitHub y Azure, así como su integración para el desarrollo y despliegue de aplicaciones en la nube. Aprenderás a utilizar GitHub para el control de versiones y a desplegar aplicaciones en Azure utilizando flujos de trabajo automatizados.

## Objetivos

- Comprender los conceptos fundamentales de GitHub.
- Aprender a crear y gestionar repositorios.
- Conocer los principios básicos de Azure.
- Desplegar una aplicación desde GitHub a Azure.
- Configurar flujos de trabajo de CI/CD utilizando GitHub Actions.

## Requisitos Previos

- Tener una cuenta en [GitHub](https://github.com/).
- Tener una cuenta en [Azure](https://portal.azure.com/).
- Visual Studio 2022 instalado.
- Conocimientos básicos de programación.

## Pasos de la Práctica

### 1. Creación de un Repositorio en GitHub

1. Accede a tu cuenta de GitHub.
2. Crea un nuevo repositorio.
3. Clona el repositorio en tu equipo local.

### 2. Subida de Código al Repositorio

1. Añade tu proyecto al repositorio local.
2. Realiza un commit de los cambios.
3. Sube los cambios a GitHub (`git push`).

### 3. Introducción a Azure

1. Accede al portal de Azure.
2. Crea un recurso (por ejemplo, una App Service).
3. Configura los parámetros básicos del recurso.

### 4. Despliegue desde GitHub a Azure

1. En Azure, vincula tu App Service con el repositorio de GitHub.
2. Configura la integración continua (CI/CD) usando GitHub Actions.
3. Realiza un push de cambios y verifica el despliegue automático.
